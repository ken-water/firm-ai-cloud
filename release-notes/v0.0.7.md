# CloudOps One v0.0.7

## Release Metadata

- Version: `v0.0.7`
- Release date: `2026-03-04`
- Release type: `minor`
- Git tag: `v0.0.7`

## Executive Summary

v0.0.7 delivers the workflow automation baseline and upgrades the web console to page-based
operations navigation, with workflow cockpit/report analytics, CSV export, and expanded RBAC
route coverage for workflow and monitoring paths.

## Highlights

- Introduced page-level console routing with active nav highlight and legacy `#section-*` compatibility.
- Added workflow templates, requests, approvals, execution APIs, and execution logs.
- Delivered workflow cockpit and report center with KPI cards, trend visuals, ranking views, and CSV export.
- Added CMDB asset stats API/dashboard and monitoring metrics API/dashboard.
- Added demo toolkit scripts for seed, health check, and cleanup workflows.

## Changed

- Console navigation now supports page routes:
  - `#/overview`, `#/cmdb`, `#/monitoring`, `#/workflow`, `#/admin`
- RBAC alias mapping expanded for nested and relative monitoring/workflow routes.
- CI policy remains manual-trigger only (`workflow_dispatch`).

## Fixed

- Workflow/monitoring route-permission mapping gaps for nested route forms.

3. Restart API and web services.

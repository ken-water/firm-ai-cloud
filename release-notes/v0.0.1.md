## Executive Summary

`v0.0.1` is the initial public open-source scaffold release of CloudOps One.
It establishes the baseline repository, architecture documents, one-click installation scripts, and runnable backend/frontend service skeletons.

## Highlights

- Initial repository bootstrap with Rust workspace and React web console.
- One-click dependency stack installation using Docker Compose.
- Offline delivery packaging and offline install scripts for customer environments.
- Initial product PRD and architecture documentation set.

## Added

## Upgrade Steps

1. Clone repository:
   - `git clone https://github.com/ken-water/firm-ai-cloud.git`
2. Install local dependencies:
   - `bash scripts/install.sh --skip-docker-install --mirror cn`
3. Start API service:
   - `cargo run -p api`
4. Start web console:
   - `cd apps/web-console && npm install && npm run dev`

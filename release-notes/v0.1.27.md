## Executive Summary

v0.1.27 focuses on perceived quality and usability acceleration. This release upgrades cockpit visual hierarchy, adds executive first-screen summary blocks, introduces configurable large-screen scene playlists, and reduces click-paths via role-aware quick actions.

## Highlights

- Unified cockpit visual system baseline with stronger hierarchy and card consistency.
- Executive summary hero blocks for fast risk/owner/action scanning.
- Large-screen scene playlist baseline with deterministic order and preview.
- Role quick actions for faster workflow entry from overview.

## Upgrade Steps

1. Pull `v0.1.27` code and restart API/web services.
2. Open `#/overview` and verify:
   - executive summary hero,
   - role quick actions,
   - large-screen scene playlist controls.
3. Validate role scene switching (`operator/admin/network/business`).
4. Run release gates:
   - `cargo check --workspace`
   - `cargo test -p api`
   - `cd apps/web-console && npm run check:ui`

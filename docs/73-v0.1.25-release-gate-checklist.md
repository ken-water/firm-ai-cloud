# v0.1.25 Release Gate Checklist

Date: 2026-03-19
Version: v0.1.25

1. API and web-console run against migrated database.

3. Demo and pilot operator accounts are available.
4. `v0.1.25` scope is mapped to strategic baseline document:

1. Planning issue maps demo narrative + KPI value + feedback closure outcomes.

## 5. Release Gates

Before publish:

```bash
cargo check --workspace
cargo test -p api
cd apps/web-console && npm run check:ui
```

Publish sync:

```bash
bash scripts/release-github-issues-check.sh --version 0.1.25 --notes-file release-notes/v0.1.25.md
make release-publish-dry VERSION=0.1.25
make release-publish VERSION=0.1.25
make release-check VERSION=0.1.25
```

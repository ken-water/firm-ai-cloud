# CloudOps One v0.1.24 Demo Scenario Pack

Date: 2026-03-19
Version: v0.1.24

## 1. Purpose

Provide a repeatable customer-demo baseline for the core value path:

1. monitoring risk signal,
2. topology risk/owner visibility,
3. handoff readiness decision,
4. AI evidence + guided actions.

## 2. Scenario Checklist

### Scenario A: First-glance cockpit review

1. Open `#/overview` and verify KPI strip and module entry map.
2. Confirm critical counts are visible in first screen.
3. Confirm at least one actionable module entry button works.

### Scenario B: Topology decision path

1. Refresh topology board.
2. Review lane grouping (`critical`, `warning`, `healthy`).
3. Verify owner state and recommended action hints.

### Scenario C: Shift handoff readiness

1. Refresh handoff readiness.
2. Confirm global readiness state (`ready/at_risk/blocking`).
3. Confirm reasons and prioritized item list are visible.

### Scenario D: AI evidence continuity

1. Run AI evidence query with monitoring preset.
2. Verify answer summary + evidence table.
3. Verify guided actions include safety boundary metadata.

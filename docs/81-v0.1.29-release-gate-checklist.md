# v0.1.29 Release Gate Checklist

Date: 2026-03-20
Version: v0.1.29

1. API and web-console run against migrated database.
2. `v0.1.28` release is complete and synchronized.
3. External pilot sample accounts are available for overview/setup flows.
4. `v0.1.29` scope is mapped to:

4. External-readiness acceptance contract is validated in release evidence.

```bash
cargo check --workspace

```

```bash

make release-publish-dry VERSION=0.1.29
```

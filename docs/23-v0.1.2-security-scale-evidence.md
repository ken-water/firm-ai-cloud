## 4. Deferred Risks for v0.1.3 Planning

- LDAP `live` connector mode is still not implemented; current delivery is dev-mode baseline plus policy validation.
- Local credential hashing currently uses SHA-256 + salt; stronger password KDF (Argon2id/scrypt) is recommended.
- SSE trend-delta continuity is partially limited by historical summary schema mismatch (missing lag fields in older runs).
- Scale evidence is from local single-node environment; multi-node and long-soak benchmarks remain pending.
